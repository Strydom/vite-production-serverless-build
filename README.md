# Overview

This is an example of a ESNext typescript vite + vite-plugin-ssr + serverless app.

## Running

### Prerequisites

- `nvm install`
- `pnpm install`

### With dev express

- `pnpm run dev:server`

### With prod express

- `pnpm run prod:server`

### With serverless-offline

- `pnpm run prod:lambda`
